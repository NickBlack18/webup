LESS - Variables

Specify global values in one place and re-use them in your stylesheet

Easy to make global changes to your CSS

LESS - Variables

LESS
            @globalBgColor: #444;

            #header {
                background-color: @globalBgColor;
            }

            #footer {
                background-color: @globalBgColor;
            }
        
Compiled CSS
            #header {
                background-color: #444;
            }

            #footer {
                background-color: #444;
            }

LESS - Variables Scope

LESS
            @globalBgColor: #444;

            #header {
                @globalBgColor: #d1d1d1;
                background-color: @globalBgColor;
            }

            #footer {
                background-color: @globalBgColor;
            }
        
Compiled CSS
            #header {
                background-color: #d1d1d1;
            }

            #footer {
                background-color: #444;
            }

LESS - Mixins

Embed all properties of a class into another class

Just like variables, but for classes

LESS - Mixins

LESS
            .baseClass {
                background-color: #fff;
                color: #444;
            }

            .childClass {
                .baseClass;
                color: #ff0000;
            }
        
Compiled CSS
            .childClass {
                background-color: #fff;
                color: #ff0000;
            }

LESS - Parametric Mixins

LESS
            .border-radius (@radius) {
                border-radius: @radius;
                -moz-border-radius: @radius;
                -webkit-border-radius: @radius;
            }

            #header {
                .border-radius(4px);
            }
        
Compiled CSS
            #header {
                border-radius: 4px;
                -moz-border-radius: 4px;
                -webkit-border-radius: 4px;
            }

LESS - Guard expressions

LESS
            .mix(@value) when (lightness(@value) >= 50%) {
                color: white;
            }

            .mix(@value) when (lightness(@value) < 50%) {
                color: black;
            }

            #header {
                .mix(#ddd);
            }
        
Compiled CSS
            #header {
                color: white;
            }

LESS - Operations

Add, substract, divide and multiply property values and colors

LESS - Operations

LESS
            @baseBgColor: #333;

            #header {
                background-color: @baseBgColor * 3;
            }
        
Compiled CSS
            #header {
                background-color: #999;
            }

LESS - Functions

Easy to manipulate values how you want

LESS - Functions

LESS
            @baseBgColor: #999;

            #header {
                background-color: darken(@baseBgColor, 30%);
            }
        
Compiled CSS
            #header {
                background-color: #666;
            }

LESS - Nested Rules

Nest selectors inside other selectors

Shorter stylesheet and clear inheritance

LESS - Nested Rules

LESS
            #header {
                .brand {
                    font-weight: bold;
                    &:hover{
                        text-decoration: underline;
                    }
                }

                .nav {
                float: right;
                }
            }
        
Compiled CSS
            #header .brand {
                font-weight: bold;
            }

            #header .brand:hover {
                text-decoration: underline;
            }

            #header .nav {
                float: right;
            }

LESS - Namespaces

LESS
            #util {
                .border-radius (@radius: 5px) {
                    border-radius: @radius;
                    -moz-border-radius: @radius;
                    -webkit-border-radius: @radius;
                }
            }

            .container {
                #util > .border-radius(15px);
            }
        
Compiled CSS
            .container {
                border-radius: 15px;
                -moz-border-radius: 15px;
                -webkit-border-radius: 15px;
            }
